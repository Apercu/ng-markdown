    • Ng-Markdown

    Based on the PageDown project,

    Ng-Markdown is an Angular directive that allow you to get a highly customizable markdown.

    It provides a <ng-markdown> element that will be replaced by a simple textarea. All you have to do is set the content attribute to bind the textarea with a var of your scope.

    The icons uses icomoon and the default style was kept to basics, but you can easily change it using associated css classes.

    Give it a try !

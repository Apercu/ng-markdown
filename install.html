    Downloads

    Here is a few ways to get all you need to get started.

    Bower

    bower install ng-markdown

    GitHub

    git clone https://github.com/Apercu/ng-markdown.git

    Wget

    Depending on your compression preferences :

    wget https://github.com/Apercu/ng-markdown/zipball/master
    wget https://github.com/Apercu/ng-markdown/tarball/master

    MouseClick

    Use this with caution and only if there is no other way to do so.

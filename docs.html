    • Ng-Markdown

    Parameters

    Name Type Description Default
    content string Ng-Model binded to the textarea. It's the only required argument. null
    sanitized boolean Sanitize the Markdown output. true
    prefix string How the elements of the markdown should be prefixed. wmd-
    postfix string How the elements of the markdown should be postfixed. When you have more than one markdown one a page, this argument is required to help the directive distinguish them. null
    customStrings object If you want to change the defaults strings. null
    preConversion function Called before conversion, text parameter is the text of the input that will be passed to the converter. 
    Parameters : text
    		 null
    postConversion function Called after conversion, text parameter is the text that have been output from the converter. 
    Parameters : text
    		 null
    postNormalization function Called after normalization, text parameter is the final result of conversion. 
    Parameters : text
    		 null
    plainLinkText function Transform the text to be displayed of bare URLs (does not affect the targets of the links). 
    Parameters : url
    		 null
    preBlockGamut function Called with the text before creating block elements like code blocks and lists. If you're creating a new kind of block structure that can include other blocks, you need to run the full block gamut again on the content of your block, for that purpose you will need to use the second parameter. Be carefull of infinite recursion. 
    Parameters : text, runBlockGamut
    		 null
    postBlockGamut function Same as preBlockGamut, but called with the text after creating block elements. 
    Parameters : text, runBlockGamut
    		 null
    preSpanGamut function Called with the text of a single block element before the span-level conversions (like bold, code spans...) 
    Parameters : text
    		 null
    postSpanGamut function Same as preSpanGamut, but called with the text after the span-level conversions. 
    Parameters : text
    		 null
    onPreviewRefresh function Called every time the editor preview has been updated. 
    Parameters : none
    		 null
    postBlockquoteCreation function Called just after the user has clicked on the Blockquote button. 
    Parameters : text
    		 null
    insertImageDialog function Function to create a new behaviour for when a user add a new image, call the callback function with your result in parameter. Must return true. 
    Parameters : callback
    		 null

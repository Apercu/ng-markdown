    • Ng-Markdown

    Events

    Name Description Usage
    refreshMarkdown Event to force the refresh of the Markdown. This is usefull when you have programmatically changed the value binded with the markdown. You can either pass a null value or '' as message to refresh all the markdowns of the page or the couple prefix + suffix to target a specific one.

    Parameters

    Name Type Description Default
    content string Ng-Model binded to the textarea. It's the only required argument. null
    sanitized boolean Sanitize the Markdown output. true
    prefix string How the elements of the markdown should be prefixed. wmd-
    suffix string How the elements of the markdown should be suffixed. When you have more than one markdown one a page, this argument is required to help the directive distinguish them. null
    custom-strings object If you want to change the defaults strings. null
    pre-conversion function Called before conversion, text parameter is the text of the input that will be passed to the converter. 
    Parameters : text
    		 null
    post-conversion function Called after conversion, text parameter is the text that have been output from the converter. 
    Parameters : text
    		 null
    post-normalization function Called after normalization, text parameter is the final result of conversion. 
    Parameters : text
    		 null
    plain-link-text function Transform the text to be displayed of bare URLs (does not affect the targets of the links). 
    Parameters : url
    		 null
    pre-block-gamut function Called with the text before creating block elements like code blocks and lists. If you're creating a new kind of block structure that can include other blocks, you need to run the full block gamut again on the content of your block, for that purpose you will need to use the second parameter. Be carefull of infinite recursion. 
    Parameters : text, runBlockGamut
    		 null
    post-block-gamut function Same as preBlockGamut, but called with the text after creating block elements. 
    Parameters : text, runBlockGamut
    		 null
    pre-span-gamut function Called with the text of a single block element before the span-level conversions (like bold, code spans...) 
    Parameters : text
    		 null
    post-span-gamut function Same as preSpanGamut, but called with the text after the span-level conversions. 
    Parameters : text
    		 null
    on-preview-refresh function Called every time the editor preview has been updated. 
    Parameters : none
    		 null
    post-blockquote-creation function Called just after the user has clicked on the Blockquote button. 
    Parameters : text
    		 null
    insert-image-dialog function Function to create a new behaviour for when a user add a new image, call the callback function with your result in parameter. Must return true. 
    Parameters : callback
    		 null
